Admin Instructions

Admin Bar

Self explanatory, but be careful of the delete button. There is no recovery or confirmation (for now).

Page or Post body text

Post and page text accepts HTML markup, and doesn't alter in any way what you input. At some point, this might be replaced with a markdown editor, but for now create your own tags.

Images

The featured image is used both in the right sidebar (top on mobile), and as the thumb on the loop page. When uploaded, it also needs a thumbnail with _thumb appended to the same filename. For example, /static/uploads/india/featured.jpg and /static/uploads/india/featured_thumb.jpg. The full size featured image needs to be able to work at just about any aspect ratio. Images used in the image gallery should also be uploaded with a thumbnail of the same size and naming schema.

Image size reference:
  • All thumbnails: 200px by 200px
  • Post featured image: 800px by 800px (being exactly square isn't important, but since it needs to work at any aspect ratio, it doesn't really help to be landscape or portrait)
  • Image gallery full-size image: 800px wide (landscape) or 600px tall (portrait)
  • Embedded images: max 800px wide, smaller for portrait. Keep the file size under 200KB

Images should be uploaded to the /static/uploads folder, then organized by country and region.

Image Gallery markup

<div class="image-gallery" data-images="/path/to/image1.jpg, /path/to/image2.jpg, etc">
</div>
